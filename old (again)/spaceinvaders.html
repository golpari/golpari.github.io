Space Invaders
Class Project
Background
As a study of object-oriented programming, I developed my own version of the old game Space Invaders as a class project. The structure and elements of this project were developed one by one, and each game element was slowly introduced into the game as I learned about each.

I've had an interest in game development, and this project was a great opportunity to explore it.
Work
I wrote this project in Processing with Java. It employed an overarching game manager class and individual classes for each element of the game (bullets, bombs, aliens, the player). The game manager class ran a constant loop to detect any updates within the gameplay without any delay in visuals.

I used bounding boxes to detect collisions between objects and subsequently adjust on-screen elements. Bounding detection also aided in trash cleanup so that any elements I could free memory by deleting any elements that were located out of the visual window.
